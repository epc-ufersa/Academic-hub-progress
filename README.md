# Academic Hub 
Sua plataforma de curadoria e revisão espaçada. Uma aplicação web projetada para centralizar o registro de materiais de estudo e facilitar a memorização ativa através de flashcards interativos.
##  Funcionalidades Principais
* **Sistema de Flashcards:** Interação de perguntas e respostas para revisão ativa.
* **Pesquisa Dinâmica:** Busca em tempo real por assuntos ou palavras-chave diretamente na biblioteca de cards.
* **Tema Claro / Escuro Automático:** O sistema detecta a preferência do sistema operacional do usuário, permitindo também a troca manual e salvamento da escolha no `localStorage`.
* **Design Responsivo:** Menu expansível (mobile-first) e layout adaptável para estudos no computador, tablet ou celular.
* **Acessibilidade:** Uso de tags semânticas e atributos `aria-label` para navegação por leitores de tela.

## Tecnologias Utilizadas
* **HTML5:** Estrutura 100% semântica.
* **CSS3:** Arquitetura com Flexbox e Grid Layout, além de transições suaves de tema.
* **JavaScript (ES6+):** Lógica modularizada (ES Modules) para controle de temas, renderização de cards e filtros de busca.
* **Tipografia e Ícones:** Google Fonts (Fraunces e Inter) e FontAwesome.

## Pré-requisitos
Por ser uma aplicação frontend estática, você não precisa de interpretadores complexos ou bancos de dados. É necessário apenas:
* Um navegador web moderno (Chrome, Firefox, Edge, Safari).
* Um editor de código como [VS Code](https://code.visualstudio.com/).
* A extensão **Live Server**.

## Instalação e Execução

Siga o passo a passo abaixo para rodar o projeto na sua máquina:

```bash
# 1. Clone este repositório
git clone [https://github.com/seu-usuario/academic-hub.git](https://github.com/seu-usuario/academic-hub.git)

# 2. Acesse o diretório do projeto
cd academic-hub

# 3. Abra o projeto no seu editor de código
code .
# 4. Inicie o Live Server no arquivo index.html. A aplicação será aberta automaticamente no seu navegador padrão (geralmente na porta http://127.0.0.1:5500). 
```
